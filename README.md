# Django Ecommerce Store 
online store using python , django , rest framework , payment and more 

## Description
Dj4-Amazon-Clone provides a comprehensive and interactive online shopping experience. Users can browse through a wide range of products, add them to their cart, and proceed to checkout to complete their purchase. The application supports user registration and login, allowing customers to manage their profiles, view order history, and track the status of their orders.

The project focuses on delivering a visually appealing and intuitive user interface, complemented by a robust backend. It emphasizes code modularity, scalability, and maintainability. The aim is to provide a solid foundation that can be extended and customized to meet specific business requirements.

## Tech Stack
The following technologies and frameworks are utilized in the development of Dj4-Amazon-Clone:

- **Django 4**: A high-level Python web framework known for its simplicity, flexibility, and scalability. Django provides a powerful toolkit for building web applications rapidly.

- **HTML5 and CSS3**: The core technologies for building the user interface. HTML5 provides the structure of the web pages, while CSS3 is responsible for the visual styling and layout.

- **JavaScript**: Used to enhance the interactivity and dynamic behavior of the application. JavaScript enables features such as client-side validation and AJAX requests.

- **Bootstrap**: A popular front-end framework that provides a collection of CSS and JavaScript components. Bootstrap streamlines the development process and ensures responsiveness across different devices.

- **SQLite**: A lightweight, serverless database management system. SQLite is used as the default database for Django development, providing a simple and efficient storage solution.

- **Amazon S3**: An object storage service provided by Amazon Web Services (AWS). Amazon S3 is used to store and serve static files such as product images, ensuring scalability and high availability.

- **Git**: A distributed version control system used for collaborative development. Git enables efficient code management, versioning, and collaboration among team members.

## Getting Started
To run Dj4-Amazon-Clone locally, follow these steps:

1. Clone the repository: `git clone https://github.com/Pythondeveloper6/Dj4-Amazon-Clone.git`

2. Install the required dependencies using `pip`:
   ````
   pip install -r requirements.txt

3. Apply the database migrations:
   ````
   python manage.py migrate

4. Start the development server:
   ````
   python manage.py runserver

5. Open your web browser and visit `http://localhost:8000` to access the application.
